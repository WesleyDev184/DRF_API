# Django Rest Framework API
Simple CRUD.

### Technologies Used:

- Django Rest Framework
- Postegres
- Python

### How To Run

Clone the repository.
Database Postegres
install Django to venv
```
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
```
Execulte the code 
```
python manage.py runserver
```
